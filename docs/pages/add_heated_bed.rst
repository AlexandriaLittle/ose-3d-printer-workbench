Centering Heated Bed to Frame & Elevating to Z Axes
---------------------------------------------------
1. Hold-down ``Ctrl`` key for selecting multiple objects
2. Select one Z axis
3. Select the Frame
4. Click the **Add Heated Bed** button on the main toolbar
